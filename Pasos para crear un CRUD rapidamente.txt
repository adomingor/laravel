🧠 Opción 1 (rápida): usar Query Builder

No necesitás modelo, simplemente cambiás el index():

use Illuminate\Support\Facades\DB;

public function index()
{
    $productos = DB::table('vw_productos_categorias')->get();

    return view('productos.index', compact('productos'));
}

---------------------------

🧠 Opción 2 (recomendada): crear un modelo para la vista

Esto es más limpio y mantenible (sobre todo para proyectos grandes).

📦 1. Crear modelo
php artisan make:model ProductoCategoriaView

📄 2. Modelo app/Models/ProductoCategoriaView.php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

/**
 * Modelo para la vista vw_productos_categorias
 */
class ProductoCategoriaView extends Model
{
    protected $table = 'vw_productos_categorias';

    protected $primaryKey = 'id';

    public $timestamps = false;

    // Es una vista → no se puede escribir
    public $incrementing = false;

    protected $guarded = [];
}

📄 3. Usarlo en el controlador
use App\Models\ProductoCategoriaView;

public function index()
{
    $productos = ProductoCategoriaView::all();

    return view('productos.index', compact('productos'));
}


🔥 Bonus PRO (paginación + búsqueda)
public function index(Request $request)
{
    $query = ProductoCategoriaView::query();

    if ($request->filled('search')) {
        $query->where('producto', 'ilike', '%' . $request->search . '%');
    }

    $productos = $query->paginate(10);

    return view('productos.index', compact('productos'));
}

⚠️ Importante
❌ No uses este modelo para:
insert
update
delete

👉 Porque es una VIEW, no una tabla
